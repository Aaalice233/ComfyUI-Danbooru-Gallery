# ComfyUI-Danbooru-Gallery 开发规范

## 代码审查循环流程

所有功能开发完成后，必须执行以下代码审查循环：

```
┌─────────────────────────────────────────────────────────┐
│                    审查循环流程                          │
├─────────────────────────────────────────────────────────┤
│  1. 运行 code-reviewer agent                            │
│           ↓                                             │
│  2. 运行 code-quality-guardian agent                    │
│           ↓                                             │
│  3. 确认发现的问题是否真实存在                           │
│           ↓                                             │
│  4. 修复确认的问题                                       │
│           ↓                                             │
│  5. 重复步骤 1-4 直到没有新问题发现                      │
└─────────────────────────────────────────────────────────┘
```
